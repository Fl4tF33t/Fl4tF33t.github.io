My Work

Here, you can explore the ongoing project I'm currently engaged in, as well as past projects, ranging from group projects to solo adventures. My responsibilities in the majority of these projects are focused on programming and implementation.

Current Work: Bugger Off! --Link To Repo-->

The current project I am working on is a tower defense game called Bugger Off! I devised Bugger Off!'s game architecture based on the GDD, including AI tower logic and troop pathfinding. I also implemented key gameplay mechanics, integrating art, animations, and logic. Additionally, I established an Excel data management system for efficient organization. Finally, I was responsible for setting up and publishing the game on to Steam.

Kitchen Chaos --Link To Repo-->

Kitchen Chaos is inspired by Overcooked, a project where I used solid coding principles and advanced techniques to improve my game development skills. This project helped me become better at game development by carefully applying best practices and methods.

GameJam: Vanha Kauna --Link To Repo-->

During my first game jam, I took the lead as the programmer for a card game called Vanha Kauna. Our collaboration extended beyond our own school, as we teamed up with another school for the competition. As first-year students, we emerged victorious in the local competition held between our classes.

Tower Toppler --Link To Repo-->

Tower Toppler came from a software engineering course where we applied the principles of a 5-day Scrum framework, we collaborated to develop a 2D platformer game. This project included creating the AI for a dynamic boss battle with various stages of attacks and behavior. I also designed a pathfinding system for the boss and supported player logic and controller functionality. Additionally, I implemented UI elements, art, and animations to enhance the game experience. Through this hands-on experience, I honed my software engineering skills through collaborative aspects of agile development methodologies.
